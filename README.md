## 🛠️ Setup Guide

<details>
<summary><b>Android &nbsp;—&nbsp; V2RayNG</b></summary>

<br>

> [!WARNING]
> <p dir="rtl">قبل از شروع یه VPN دیگه روشن کن</p>

<p dir="rtl">
1. دانلود و نصب V2RayNG<br>
2. منو <code>☰</code> بالا چپ ← <code>Subscription Group Setting</code><br>
3. دکمه <code>+</code> بالا راست ← لینک ساب رو پیست کن ← ذخیره<br>
4. سه‌نقطه بالا راست ← <code>Update Subscription</code><br>
5. یه سرور انتخاب کن ← دکمه وسط ← وصل شدی ✅
</p>

</details>

<details>
<summary><b>Android &nbsp;—&nbsp; Happ</b></summary>

<br>

> [!WARNING]
> <p dir="rtl">قبل از شروع یه VPN دیگه روشن کن</p>

<p dir="rtl">
1. دانلود و نصب Happ<br>
2. پایین صفحه ← <code>Subscribe</code><br>
3. لینک ساب رو پیست کن ← <code>Import</code><br>
4. یه سرور انتخاب کن ← وصل شو ✅
</p>

</details>

<details>
<summary><b>iOS &nbsp;—&nbsp; Streisand</b></summary>

<br>

> [!WARNING]
> <p dir="rtl">قبل از شروع یه VPN دیگه روشن کن</p>

<p dir="rtl">
1. دانلود و نصب Streisand<br>
2. پایین صفحه ← دکمه <code>+</code><br>
3. گزینه <code>Subscribe</code> ← لینک ساب رو پیست کن ← <code>OK</code><br>
4. یه سرور انتخاب کن ← وصل شو ✅
</p>

</details>

<details>
<summary><b>Windows &nbsp;—&nbsp; V2RayN</b></summary>

<br>

> [!WARNING]
> <p dir="rtl">قبل از شروع یه VPN دیگه روشن کن</p>

<p dir="rtl">
1. دانلود <code>v2rayN-With-Core.zip</code> ← اکسترکت ← اجرای <code>v2rayN.exe</code><br>
2. منو بالا ← <code>Subscriptions</code> ← <code>Subscription Group Setting</code><br>
3. دکمه <code>+</code> ← لینک ساب رو توی فیلد <code>URL</code> بذار ← <code>OK</code><br>
4. منو ← <code>Subscriptions</code> ← <code>Update Subscription</code><br>
5. کلیک راست روی سرور ← <code>Set as Active Server</code> ← وصل شدی ✅
</p>

</details>
